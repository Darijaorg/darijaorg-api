Darija Words API Documentation

Introduction

The Darija Words API provides access to a dataset of Darija (Moroccan Language) words, including nouns, verbs, adjectives, adverbs, and pronouns. The API allows you to retrieve, search, and filter words based on various criteria.

Base URL: https://darijaorg-api.vercel.app

Endpoints

1. Get All Words

Endpoint: /words

Method: GET

Description: Retrieve all words in the dataset.

Example Request:

GET https://darijaorg-api.vercel.app/words

4. Search Words

Endpoint: /search

Method: GET

Query Parameters:

  • q: Search query (e.g., brother).

Error Handling

The API returns the following error responses:

  • 404 Not Found: The requested resource does not exist.
  • 400 Bad Request: Invalid query parameters or malformed request.

Rate Limiting

The API is rate-limited to 100 requests per hour per IP address. Exceeding this limit will result in a 429 Too Many Requests response.
